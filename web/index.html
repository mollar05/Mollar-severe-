🔌 window.TeachableTextAI — API
predict(text)
→ Promise<{ label, confidence, scores }>
텍스트를 분류합니다. 학습/로드 완료 후 사용 가능. confidence는 0~1 소수, scores는 전체 클래스 확률 객체.
onReady(callback)
→ void · callback({ labels })
학습 또는 모델 로드가 완료되면 자동 호출. 이미 준비된 경우 즉시 실행.
isReady()
→ boolean
지금 바로 예측 가능한지 확인.
getLabels()
→ string[]
현재 등록된 클래스 레이블 목록 반환.
// ① 학습 완료 후 자동 실행 TeachableTextAI.onReady(({ labels }) => { console.log('준비됨, 클래스:', labels); }); // ② 직접 예측 호출 const result = await TeachableTextAI.predict("안녕하세요"); console.log(result.label); // "인사" console.log(result.confidence); // 0.97 console.log(result.scores); // { 인사: 0.97, 음식: 0.03 } // ③ 준비 여부 확인 후 사용 if (TeachableTextAI.isReady()) { const { label, confidence } = await TeachableTextAI.predict(inputText); }
